Data in hibernate must be paginated. Only load data in memory when requested by the user who wishes to view it.

Noncompliant Code Example

     public List getList() {
         Criteria criteria = session.createCriteria(Foo.class);
         List result = criteria.list();
         return result;
     }

Compliant Solution

     public List getFirstPage() { //Compliant
         Criteria criteria = session.createCriteria(Foo.class);
         criteria.setFirstResult(FIRST_RESULT);
         criteria.setMaxResults(PAGE_SIZE);
         List firstPage = criteria.list();
         return firstPage;
     }
